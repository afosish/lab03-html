# lab03-html
У цій практичній роботі я створив базову структуру вебсайту на HTML5 що складається з трьох сторінок. На головній сторінці розроблено каркас із використанням семантичних тегів таких як header nav main та footer. Додано текстовий контент списки зображення з підписом та таблицю з розкладом занять.

На сторінці про мене реалізовано роботу з мультимедіа. Я додав відео та аудіо плеєри з підтримкою декількох форматів файлів та альтернативним текстом для завантаження. Також на цій сторінці підключено зовнішній скрипт на джаваскрипт який додає інтерактивність кнопці змінюючи її текст при натисканні.

Сторінка з контактами містить форму зворотного зв'язку з різними типами полів серед яких текстове введення електронна пошта номер телефону випадаючий список радіокнопки та чекбокс. Для всіх полів налаштована правильна прив'язка лейблів та додана базова валідація даних через атрибути мови розмітки.
