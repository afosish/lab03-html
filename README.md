# lab03-html

У цій практичній роботі я розширив базову структуру вебсайту на HTML5, додавши повноцінне CSS-оформлення. Проєкт складається з трьох сторінок, об'єднаних спільним дизайном за допомогою зовнішніх таблиць стилів (`style.css` та `responsive.css`). На головній сторінці розроблено семантичний каркас, який тепер має обмежений контейнер для контенту, стилізовані картки та адаптивні зображення.

Реалізовано глобальну дизайн-систему: задано CSS-змінні для кольорової палітри (бежево-кавові відтінки), налаштовано типографіку, відступи та скидання базових стилів. Для побудови макетів активно використовуються сучасні технології Flexbox (для шапки та навігації) та CSS Grid. Зокрема, стандартний HTML-тег таблиці було повністю замінено на гнучку структуру на базі `div` та CSS Grid для кращої адаптивності та контролю над дизайном.

На сторінці «Про мене» реалізовано роботу з мультимедіа. Відео та аудіо плеєри акуратно вписані в сітку карток. Також збережено роботу зовнішнього скрипта на JavaScript, який додає інтерактивність кнопці (яка тепер має плавні CSS-переходи та ефекти при наведенні).

Сторінка з контактами містить форму зворотного зв'язку, яку було повністю стилізовано: додано тіні, скруглення, акцентні рамки при фокусі (`:focus`) для полів введення, випадаючих списків та текстової області. Кнопки мають стани `:hover` для покращення користувацького досвіду. 

Увесь сайт є повністю адаптивним (Responsive Web Design). Завдяки медіазапитам (`@media queries`) макет сторінок, навігація, розміри шрифтів та відступи автоматично підлаштовуються під розміри екранів мобільних телефонів та планшетів.


**JavaScript та Інтерактивність**
Додано зовнішній файл main.js, який забезпечує наступний функціонал:
Динамічна навігація: Підсвічування активної сторінки в меню навігації.
Мобільне меню: Кнопка-бургер для відкриття/закриття меню на малих екранах.
Темна тема: Перемикач світлої/темної теми зі збереженням вибору користувача у localStorage.
UI Елементи: Акордеон для розділу FAQ, динамічний фільтр для медіа-контенту, модальне вікно (lightbox) для перегляду зображень, кнопка "Повернутися вгору" та автооновлення року у футері.
Покращена форма (Контакти):Клієнтська валідація даних (перевірка довжини імені, формату email, непорожнього повідомлення).
Збереження введених даних (чернетки) у localStorage під час набору тексту.
Лічильник введених символів для текстової області.
Обробка події submit зі збором даних через FormData та імітацією відправлення повідомлення (без перезавантаження сторінки).
